# 邮箱验证功能说明

## 已接入能力

- 注册账号必须校验邮箱验证码，场景为 `REGISTER`。
- 登录支持两种方式：密码登录、邮箱验证码登录，验证码场景为 `EMAIL_LOGIN`。
- 忘记密码/重置密码必须校验邮箱验证码，场景为 `RESET_PASSWORD`。
- 登录后修改密码必须同时校验旧密码和邮箱验证码，场景为 `CHANGE_PASSWORD`。
- 新注册和新修改的密码使用 BCrypt 存储；旧 MD5 密码在用户密码登录成功后会自动升级为 BCrypt。

## 邮箱配置

当前使用 163 邮箱 SMTP：

```properties
spring.mail.host=smtp.163.com
spring.mail.port=465
spring.mail.username=15597812468@163.com
spring.mail.password=${GUIZHOU_JOB_MAIL_PASSWORD:你的163授权码}
spring.mail.properties.mail.smtp.auth=true
spring.mail.properties.mail.smtp.ssl.enable=true
spring.mail.properties.mail.smtp.starttls.enable=false

app.email-code.expire-minutes=5
app.email-code.send-interval-seconds=60
```

建议正式提交或部署时，把授权码放到环境变量 `GUIZHOU_JOB_MAIL_PASSWORD`，不要长期明文写在配置文件里。

## 建表 SQL

```sql
CREATE TABLE IF NOT EXISTS email_verification_code (
  id BIGINT PRIMARY KEY AUTO_INCREMENT,
  email VARCHAR(100) NOT NULL COMMENT '接收验证码的邮箱',
  scene VARCHAR(40) NOT NULL COMMENT 'REGISTER/EMAIL_LOGIN/RESET_PASSWORD/CHANGE_PASSWORD等',
  code_hash VARCHAR(128) NOT NULL COMMENT '验证码哈希值',
  expire_time DATETIME NOT NULL COMMENT '过期时间',
  used TINYINT DEFAULT 0 COMMENT '是否已使用',
  send_ip VARCHAR(64) NULL COMMENT '发送请求IP',
  create_time DATETIME DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
  used_time DATETIME NULL COMMENT '使用时间',
  INDEX idx_email_scene (email, scene),
  INDEX idx_expire_time (expire_time)
);
```

## 接口

```text
POST /email/code/send
body: { "email": "user@163.com", "scene": "REGISTER" }

POST /user/register
body: { "nickname": "...", "email": "...", "password": "...", "emailCode": "123456" }

POST /user/login
body: { "email": "...", "password": "..." }

POST /user/login/code
body: { "email": "...", "emailCode": "123456" }

POST /user/password/reset
body: { "email": "...", "emailCode": "123456", "newPassword": "..." }

POST /user/password/change
header: X-User-Id
body: { "oldPassword": "...", "newPassword": "...", "emailCode": "123456" }
```
